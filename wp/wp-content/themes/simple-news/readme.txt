== Description ==
Simple News is a simple WordPress theme based on Bootstrap front-end framework, suitable for news or blog site.
Simple News theme is a complete solution for whatever contents you have.
It is as solid as the Bootstrap itself. All features of Bootstrap are supported by this theme.
Simple News theme is optimized for readability, highly mobile friendly and with neat and balanced layout. It is minimal, lightweight, yet multipurpose.
This release is based on Bootstrap version 4.5.2 which is the latest stable version.
This theme also contains Fork Awesome icon font, a fork of the renowned Font Awesome icon, enabling you to insert hundreds of icons easily.
Note: Simple News theme only support 2 levels menus (dropdown only, no submenus beyond).


== Frequently Asked Questions ==

= How to set-up or modify content appearance? =
Simple News theme is built on top of Bootstrap 4.5.2. Please refer to Bootstrap documentation to set up or modify your content.
https://getbootstrap.com/docs/

= How to change font? =
1. Go to Appearance > Customize
2. Add your custom CSS style, in this case, font-family for body
   Example:
   body {
     font-family: Roboto, Arial, sans-serif;
   }

= How to add icon to menu? =
1. Go to "Appearance" > "Menus"
2. Click "Screen Options" on top right
3. Select "CSS Classes"
4. Click "down arrow" on right edge of the menu which will be added with icon
5. On "CSS Classes (optional)" add this code:
   fa fa-###### fa-fw
   ###### is the icon name as listed in https://forkaweso.me/Fork-Awesome/icons/
