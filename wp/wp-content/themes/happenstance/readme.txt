== Description ==

HappenStance is an easily customizable theme that can be used for your blog, magazine, business, eCommerce or events website. It is a fully responsive and Retina ready theme that allows for easy viewing on any device.

== Installation ==

You can install this theme through the WordPress installer under "Appearance > Themes - Add New" by searching for it.
Alternatively, you can download the ZIP file, unzip it and move the extracted folder "happenstance" via FTP to the "wp-content/themes" directory on your web server. Then you will be able to activate the theme through "Appearance > Themes".
