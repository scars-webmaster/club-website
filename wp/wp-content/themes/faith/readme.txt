== Description ==

Faith is a free responsive WordPress theme for churches. It works well with page builders like Elementor, Beaver Builder, Visual Composer, Brizy, Divi, SiteOrigin, etc. Up to 10 pages can be featured on the homepage, 5 in the slideshow and 5 below it. 


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= How to setup the front page like the demo site? =

= Footer Area =

The footer area contains 5 widgetized columns of equal width.

== Quick Specs (all measurements in pixels) ==

	1. The main column width is 680.
	2. In full-width layout the content width is 980.
	3. A widget in the sidebar is 250.
	4. The featured image on the front page and on pages works best with images at least 1600 wide.
	5. Featured Images for posts should be at least 240 wide.
	6. The footer widget columns are of equal width: 200.
